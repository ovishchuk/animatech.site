# Animatech Brand Colors

## Фірмові кольори Animatech

Наша палітра складається з трьох основних неонових кольорів, що символізують живі технології:

### 🎨 Основні кольори

- **Neon Cyan**: `#00ffff` - чиста енергія та інновації
- **Neon Magenta**: `#ff00ff` - творчість та креативність  
- **Neon Accent**: `#00ccaa` - гармонія та баланс

### 🔧 CSS змінні

```css
:root {
    --neon-cyan: #00ffff;
    --neon-magenta: #ff00ff;
    --neon-accent: #00ccaa;
}
```

### 🌈 Градієнти

Основний градієнт бренду:
```css
background: linear-gradient(90deg, #00ffff, #ff00ff, #00ccaa);
```

### 💡 Філософія кольорів

- **Cyan** - символізує чистоту коду та технологічну досконалість
- **Magenta** - представляє творчий підхід та людський аспект технологій
- **Accent** - відображає баланс між логікою та емоціями в живих системах

Ці три кольори створюють унікальну ідентичність Animatech та використовуються в усіх матеріалах бренду.
