# Робота з логотипом Animatech

## Завдання
Створити логотип з прозорим фоном для сайту Animatech

## Файли
- Оригінал: logo.jpg (в корені проекту)
- Робочі файли: /tmp/
- Фінальний: html/logo.png

## Проблема
ImageMagick конвертує PNG в 1-bit grayscale при роботі з масками, втрачаючи прозорість.

## Рішення
Потрібно знайти правильний спосіб створення PNG з прозорим фоном та кольоровим візерунком.

## Кроки
1. Скопіювати оригінал в tmp
2. Експериментувати з різними методами
3. Перевіряти результат
4. Копіювати фінальний файл в html/

## Статус
- ✅ Папка tmp доступна
- ✅ Оригінал знайдено
- 🔄 Пошук правильного методу...

## Експерименти
Метод 1: -transparent white
Метод 2: -negate -transparent black  
Метод 3: Маски з alpha каналом
Метод 4: -alpha off -compose copy
